# ReadSlot Privacy Policy

ReadSlot stores saved URLs, titles, notes, tags, estimates, settings, and reading history locally
in the browser. It does not operate a server, sell data, display advertising, or collect
analytics.

When a user connects Google Calendar, ReadSlot requests calendar-list metadata, free/busy
availability, and permission to manage events. Calendar availability is processed in the
extension. Saved URLs are sent to Google only when the user explicitly confirms a reading
event whose description contains those URLs.

Google access tokens are managed by Chrome Identity and are never written to ReadSlot storage.
Users can disconnect Google, export local data, or permanently delete all local ReadSlot data.
