# ADR 0003: ReadSlot Product Name

Status: Accepted, 2026-07-13.

The product name is **ReadSlot**, with the tagline **Schedule what you save.** The name directly
connects reading with reserving a time slot and is clearer in a browser-extension listing than
the former working name. The rename occurred before OAuth configuration or publication,
so package, storage, backup, Calendar metadata, release, and internal namespaces use `readslot`
without a legacy compatibility layer.

A preliminary collision search found no obvious competing reading product under the exact name.
Formal trademark, Chrome Web Store, GitHub, and domain clearance remains a publication gate.
